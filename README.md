V1.0.0
✅ 核心功能：
1. 自动获取当前日期作为文件夹名前缀（格式：YYYY-MM-DD_）
2. 支持多种时间输入格式智能转换：
   • 1416    → 14_16
   • 905     → 09_05  
   • 14_16   → 保持 14_16
3. 输入项目名称，组合生成完整文件夹名：
   • 示例：2026-03-18_14_16-项目汇报
4. 一键复制结果到剪贴板（兼容新旧浏览器）
5. 基础表单验证：
   • 项目名称不能为空
   • 时间格式校验（小时00-23，分钟00-59）

✅ 交互体验：
• 结果区域默认隐藏，生成后显示
• 复制按钮动态显示
• 输入框有 placeholder 提示

✅ 技术实现：
• 纯前端 HTML+CSS+JS，无依赖
• 响应式布局，最大宽度 600px 居中
• 使用 Microsoft YaHei 字体，适配中文环境
